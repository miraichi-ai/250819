4つのSub Agent構成

📋 今回構築する4つの専門エージェント

それぞれが異なる専門分野を担当し、連携してX投稿を自動生成するシステムを構築します。

1

🔍 リサーチエージェント

役割：X投稿用の調査・分析・材料収集を担当するAI

📝 主な機能
  • ユーザー指定テーマの深掘り調査
  • 類似アカウントの投稿傾向分析
  • トレンドキーワードの抽出
  • 情報収集結果のレポート作成
⚠️ 制限事項：ブラウザログインが必要な情報は直接取得できません。Playwright MCPなどの追加ツールが必要です。
2

📊 過去投稿分析エージェント

役割：伸びている投稿のパターンを分析し、成功要因を抽出

📝 分析項目
  • フックパターン（読者の注意を引く書き出し）
  • ボディパターン（本文の構成方法）
  • エンゲージメントブースター要素
  • 文章の癖・特徴の抽出
エンゲージメントブースター = いいね・リツイート・コメントなどの反応を促進する投稿要素。
3

⚙️ システムプロンプト生成エージェント

役割：分析結果からジャンル別の投稿生成ルールを作成

📝 生成内容
  • 技術AI系投稿用プロンプト
  • ニュース・情報発信系プロンプト
  • エンゲージメント重視プロンプト
  • 個人ブランディング系プロンプト
4

✍️ 投稿作成エージェント

役割：最終的な投稿文を生成し、複数パターンを提案

📝 出力内容
  • メイン投稿文（完成版）
  • 代替パターン（2-3種類）
  • 適切なハッシュタグ提案
  • 投稿タイミング推奨

📋 実際に手を動かして学ぶ部分

講座参加者と一緒に、実際にClaude Code環境を構築し、Sub Agentを作成して投稿生成まで体験します。

🛠️ Step 1: 開発環境の準備

📋 必要なツール・アカウント
  • GitHubアカウント：コード管理とCodespaces利用
  • Claude API：エージェント実行のため
  • Node.js環境：開発実行環境
# リポジトリ作成とクローン git clone [リポジトリURL] cd x-post-agent # Node.js環境確認・更新 npm install npm update # Claude Code インストール npm install -g claude-code
GitHub Codespaces = ブラウザ上で動作する開発環境。ローカルにNode.js環境がない場合でも、オンラインで開発作業ができる。

🔐 Step 2: Claude Code認証設定

📋 認証プロセス
  1. Claude Code起動時に認証画面が表示
  2. 「承認する」ボタンをクリック
  3. 表示されたコードをコピー
  4. ターミナルに貼り付けて認証完了
💡 認証方法：APIキーまたはサブスクリプション認証の2通りが選択可能です。

📝 Step 3: claude.md作成

📋 プロジェクト定義ファイル

Claude Codeが理解できるマークダウン形式で、プロジェクト全体の概要と目的を定義します。

# claude.md の基本構造 # X投稿マルチエージェントシステム ## プロジェクト概要 4つの専門エージェントが連携して、高品質なX投稿文を自動生成するシステム ## エージェント構成 1. リサーチエージェント 2. 過去投稿分析エージェント 3. システムプロンプト生成エージェント 4. 投稿作成エージェント

🤖 Step 4: Sub Agent作成

📋 エージェント作成コマンド

Claude Codeのagentsコマンドを使用して、4つの専門エージェントを順次作成します。

# エージェント作成の流れ /agents create-new-agent # プロジェクト選択 project # エージェント名入力 x-research-agent # claude.mdの内容を貼り付け [claude.mdの全文をコピー&ペースト] # モデル選択（推奨：Opus） opus
✅ 講座での実績：4つのエージェントを約10分で作成完了！Claude Codeの高速性を実感できました。

「ついすぽ」を使った過去投稿分析

📋 そらさんが実演した投稿データ収集方法

「ついすぽ -Tweet Export-」Chrome拡張機能を使って、実際の投稿データを収集し、分析エージェントに読み込ませる手順を解説。

🌐 ついすぽインストールと使用方法

📋 データ収集手順
  1. Chrome ウェブストアから「ついすぽ」をインストール
  2. Xで分析したいアカウントを検索（例：from:ユーザー名）
  3. 「100いいね以上」など条件を追加して絞り込み
  4. F12で開発者ツールを開き、「ついすぽ」タブを選択
  5. スクロールしてデータ件数を増やす
  6. 「Download」でCSVファイルを取得
⚠️ 重要：ついすぽは検索結果からのみ使用可能。直接プロフィール画面からは使用できません。

📊 Googleスプレッドシートでの整理

📋 データ整理プロセス
  1. Googleスプレッドシートを新規作成
  2. 「ファイル」→「インポート」でCSVアップロード
  3. 不要な列（URL、メタデータ等）を削除
  4. 投稿テキストのみを抽出
  5. 改行・空白を整理してクリーンなデータに

🔍 分析エージェントでの解析実行

📋 そらさんの投稿から抽出された特徴
  • フックパターン：驚きと興奮を伝える書き出し
  • エモーショナル要素：親しみやすい口調
  • プロブレムソルビング：問題解決型の構成
  • CTA（Call To Action）：スレッド形式の活用
  • ストラクチャー：空白・改行を効果的に使用
CTA（Call To Action） = 読者に具体的な行動を促す要素。「リツイートお願いします」「続きはスレッドで」など。
